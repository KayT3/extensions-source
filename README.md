# Mihon Novel Extensions

Unofficial Mihon extension repository for **novel sources**.

This is a personal fork of the Keiyoushi extensions project with manga sources removed and modifications for novel support.

## Add the Repository

Add the following URL in Mihon:

```
https://ext.kayt.cloud/index.min.json
```

Steps:

1. Open **Mihon**
2. Go to **Browse → Extensions**
3. Tap **⋮ → Extension Repositories**
4. Add the URL above

## Notes

* Extensions use a **third-party API** to generate novel chapter images.
* That API is written in **C#** and is available publicly on my GitHub profile.
* This repository is mainly for **personal use and experimentation**.
